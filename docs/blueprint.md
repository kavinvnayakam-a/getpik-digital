# **App Name**: GetPik Agency

## Core Features:

- Portfolio Showcase: Display case studies of completed projects with images, descriptions, and client testimonials.
- SEO Analysis Tool: Analyze the SEO of the agencies website and suggest changes.
- Enquiry Form: Collect visitor inquiries through a customizable form with fields for name, email, project details, and budget.
- Form Submission & Management: Securely handle form submissions and allow the admin to manage submissions.
- Client Management: Helps organize clients into leads, current clients, and completed. Allow you to take quick notes for the clients too. You can drag and drop client categories to switch.
- Content Generation Tool: A tool to help brainstorm content for blog posts, website descriptions, and client outreach, incorporating industry keywords.

## Style Guidelines:

- Primary color: A vibrant purple (#9F79EE) to convey creativity and innovation.
- Background color: A desaturated bluish-purple (#E0E5F7) to provide a calming backdrop.
- Accent color: A vivid magenta (#E079EE) to highlight calls to action and key information.
- Body and headline font: 'Space Grotesk' sans-serif for a contemporary and readable feel, paired with 'Inter' for body text where blocks of text exceed one paragraph in length.
- Use a consistent set of outline icons for navigation and representing services.
- Employ a clean, grid-based layout with generous whitespace for readability and visual appeal.
- Incorporate subtle animations and transitions on page load and when interacting with elements.
